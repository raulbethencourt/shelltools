# wcss

Examples and implementations from the "Wicked Cool Shell Scripts" book by Dale Dougherty & Arnold
Robbins. Organized by chapters for easy reference and learning.

## Chapters

### Chapter 1: Basics

- **ansicolor.sh** - ANSI color utilities
- **echon.sh** - Echo without newline
- **filelock.sh** - File locking mechanisms
- **inpath.sh** - Path validation
- **nicenumber.sh** - Number formatting
- **normdate.sh** - Date normalization
- **scriptbc.sh** - Calculator with bc
- **test_library.sh** - Testing utilities
- **validalnum.sh** - Alphanumeric validation
- **validdate.sh** - Date validation
- **validfloat.sh** - Float validation
- **validint.sh** - Integer validation

### Chapter 2: File Management

- **DIR.sh** - Directory utilities
- **fmt.sh** - Text formatting
- **formatdir.sh** - Directory formatting
- **hmcmds.sh** - Hidden command helpers
- **locate.sh** - File location tools
- **logrm.sh** - Logged removal
- **mklocatedb.sh** - Locate database creation
- **mknvfdb.sh** - NVF database tools
- **newrm.sh** - Enhanced removal
- **timein.sh** - Time tracking
- **unrm.sh** - Undo removal

### Chapter 3: Productivity

- **addagenda.sh** - Add agenda items
- **agenda.sh** - Agenda management
- **calc.sh** - Calculator
- **convertatemp.sh** - Temperature conversion
- **loancalc.sh** - Loan calculations
- **remember.sh** - Note taking
- **remindme.sh** - Reminders

### Chapter 4: System Utilities

- **bestcompress.sh** - Compression optimization
- **showfile.sh** - File display tools
- **toolong.sh** - Long filename handling

### Chapter 5: Advanced Tools

- **diskhogs.sh** - Disk usage analysis
- **fquota.sh** - File quota management
- **validator.sh** - General validation tools
